# Frontend 依赖安装说明

## 问题原因
Windows 系统路径长度限制（MAX_PATH 260字符）导致 npm/pnpm 安装失败。

## 解决方案

### 方案1：启用 Windows 长路径支持（推荐）

1. 以管理员身份运行 PowerShell
2. 执行以下命令启用长路径支持：
```powershell
New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force
```
3. 重启电脑
4. 在 frontend 目录执行：
```bash
npm install --registry=https://registry.npmmirror.com
```

### 方案2：使用 Yarn（更简单）

1. 安装 Yarn（如果未安装）：
```bash
npm install -g yarn
```

2. 在 frontend 目录执行：
```bash
yarn install --registry=https://registry.npmmirror.com
```

### 方案3：移动项目到更短的路径

将项目移动到路径更短的位置，例如：
- `C:\project\volunteer`
- `D:\vs`

然后再执行安装命令。

## 验证安装

安装成功后，运行：
```bash
npm run dev
```

如果能正常启动开发服务器，说明安装成功。
