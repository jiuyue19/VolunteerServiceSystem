# 收货地址管理功能使用说明

## 🎯 功能概述

收货地址管理页面提供了完整的地址管理功能，包括：
- ✅ 地址列表展示
- ✅ 添加/编辑地址
- ✅ 批量管理模式
- ✅ 高德地图选址
- ✅ 默认地址设置

## 🔧 配置要求

### 1. 高德地图API密钥配置

在使用地图选址功能前，需要配置高德地图API密钥：

1. **申请API密钥**：
   - 访问 [高德开放平台](https://console.amap.com/dev/key/app)
   - 注册账号并创建应用
   - 获取Web服务API密钥

2. **配置密钥**：
   ```javascript
   // 编辑文件：frontend/src/config/amap.js
   export const AMAP_CONFIG = {
     key: '您的高德地图API密钥', // 替换这里
     version: '1.4.15',
     plugins: [
       'AMap.Geocoder',
       'AMap.PlaceSearch',
       'AMap.AutoComplete'
     ]
   }
   ```

## 📱 功能使用指南

### 1. 基础地址管理

#### **查看地址列表**
- 页面自动加载所有收货地址
- 显示联系人、电话、完整地址
- 默认地址有绿色"默认"标签

#### **添加新地址**
1. 点击右上角"添加新地址"按钮
2. 填写表单信息：
   - 联系人姓名（必填，2-20字符）
   - 联系电话（必填，手机号格式）
   - 所在地区（省市区三级联动）
   - 详细地址（必填，5-100字符）
   - 地址标签（可选：家/公司/学校）
   - 是否设为默认地址

#### **编辑地址**
1. 点击地址卡片右侧"编辑"按钮
2. 修改相应信息
3. 点击"更新"保存

#### **设置默认地址**
- 点击"设为默认"按钮
- 系统自动取消其他地址的默认状态

### 2. 批量管理模式

#### **进入管理模式**
1. 点击"管理"按钮
2. 地址卡片左侧出现复选框
3. 头部显示批量操作按钮

#### **批量选择**
- **单选**：点击地址卡片左侧复选框
- **全选**：点击头部"全选"复选框
- **部分选择**：全选复选框显示为半选状态

#### **批量删除**
1. 选择要删除的地址
2. 点击"删除选中"按钮
3. 确认删除操作

**注意**：
- 默认地址不能被删除
- 如果选中包含默认地址，会提示先设置其他地址为默认

#### **退出管理模式**
- 点击"取消"按钮
- 自动清除所有选择状态

### 3. 地图选址功能

#### **打开地图选址**
1. 在添加/编辑地址对话框中
2. 点击详细地址下方的"地图选址"按钮

#### **搜索地址**
1. 在搜索框输入地址关键词
2. 点击搜索按钮或按回车键
3. 地图自动定位到搜索结果

#### **点击选址**
1. 在地图上点击任意位置
2. 系统自动进行逆地理编码
3. 显示选中地址的详细信息

#### **确认选择**
1. 选择地址后点击"确认选择"
2. 地址信息自动填充到表单
3. 包括省市区和详细地址

## 🔍 故障排除

### 地图无法加载
1. **检查API密钥**：确保已正确配置高德地图API密钥
2. **检查网络**：确保能访问高德地图API服务
3. **检查控制台**：查看浏览器控制台是否有错误信息

### 地址搜索失败
1. **关键词优化**：使用更具体的地址描述
2. **网络检查**：确保网络连接正常
3. **API配额**：检查高德地图API调用次数是否超限

### 表单验证错误
1. **联系人姓名**：确保2-20个字符
2. **联系电话**：确保是有效的手机号格式
3. **地区选择**：确保选择了省份和城市
4. **详细地址**：确保5-100个字符
