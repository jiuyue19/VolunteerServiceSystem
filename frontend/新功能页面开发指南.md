# 新功能页面开发指南

## 📋 头像下拉菜单功能列表

已为头像下拉菜单添加了以下功能选项：

### ✅ 已实现
- **个人信息** - 跳转到 `/volunteer/profile`
- **我的帖子** - 跳转到 `/volunteer/my-posts`
- **我的评论** - 跳转到 `/volunteer/my-comments`
- **我的兑换** - 跳转到 `/volunteer/my-exchanges`
- **我的订单** - 跳转到 `/volunteer/my-orders`
- **收货地址** - 跳转到 `/volunteer/address-book`
- **修改密码** - 跳转到 `/volunteer/change-password`
- **退出登录** - 清除登录信息并跳转到登录页

## 🛠️ 开发新功能页面步骤

### 1. 创建页面文件
基于 `PageTemplate.vue` 模板创建新页面：

```bash
# 复制模板文件
cp PageTemplate.vue MyPosts.vue
cp PageTemplate.vue MyComments.vue
cp PageTemplate.vue MyExchanges.vue
cp PageTemplate.vue MyOrders.vue
```

### 2. 修改页面内容
根据功能需求修改页面内容：
- 更新页面标题
- 设计数据表格结构
- 实现相应的API调用
- 添加特定的业务逻辑

### 3. 添加路由配置
在 `router/index.js` 中添加新路由：

```javascript
{
  path: '/volunteer/my-posts',
  name: 'MyPosts',
  component: () => import('@/views/volunteer/MyPosts.vue'),
  meta: { requiresAuth: true }
},
{
  path: '/volunteer/my-comments', 
  name: 'MyComments',
  component: () => import('@/views/volunteer/MyComments.vue'),
  meta: { requiresAuth: true }
},
{
  path: '/volunteer/my-exchanges',
  name: 'MyExchanges', 
  component: () => import('@/views/volunteer/MyExchanges.vue'),
  meta: { requiresAuth: true }
},
{
  path: '/volunteer/my-orders',
  name: 'MyOrders',
  component: () => import('@/views/volunteer/MyOrders.vue'), 
  meta: { requiresAuth: true }
}
```

### 4. 更新Layout.vue中的路由跳转
将 `Layout.vue` 中对应的 `ElMessage.info` 替换为实际路由跳转：

```javascript
case 'my-posts':
  router.push('/volunteer/my-posts')
  break
case 'my-comments':
  router.push('/volunteer/my-comments')
  break
case 'my-exchanges':
  router.push('/volunteer/my-exchanges')
  break
case 'my-orders':
  router.push('/volunteer/my-orders')
  break
```

## 🎨 样式规范

所有新页面请遵循以下样式规范：

### 颜色主题
- 主色调：`#ce4c4c`
- 悬停色：`#d97373`
- 文字色：`#333`
- 辅助色：`#606266`

### 布局规范
- 页面容器：`padding: 20px`
- 卡片最大宽度：`max-width: 1200px`
- 按钮间距：`gap: 10px`
- 分页居中对齐

### 组件使用
- 使用 Element Plus 组件库
- 表格使用 `border stripe` 属性
- 按钮使用统一的主题色
- 加载状态使用 `v-loading`

## 🔧 API开发建议

为每个功能页面创建对应的API接口：

### 我的帖子
```javascript
// api/forum.js
export function getMyPosts(params) {
  return request({
    url: '/forum/my-posts',
    method: 'get',
    params
  })
}
```

### 我的评论

## 📝 开发注意事项

1. **权限控制**：所有页面都需要登录权限
2. **错误处理**：统一使用 `ElMessage` 显示错误信息
3. **加载状态**：数据加载时显示loading效果
4. **响应式设计**：确保在不同屏幕尺寸下正常显示
5. **数据验证**：对用户输入进行适当验证
6. **SEO优化**：为页面添加合适的title和meta信息

## 🚀 快速开始

1. 复制 `PageTemplate.vue` 作为新页面的起点
2. 根据功能需求修改页面内容
3. 添加路由配置
4. 更新 `Layout.vue` 中的跳转逻辑
5. 测试页面功能是否正常

完成以上步骤后，新功能页面就可以通过头像下拉菜单访问了！
